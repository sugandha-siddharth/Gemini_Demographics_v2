# Gemini Demographics v2

This is a **React + Vite + Tailwind** project (not Next.js).

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy on Vercel
- Framework Preset: **Vite**
- Build Command: `npm run build`
- Output Directory: `dist`

> Note: Your `App.jsx` currently contains a hardcoded Census API key as provided in the source document. 
> If you prefer, you can move it into Vercel Environment Variables later.
